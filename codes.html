Codes
 

Codes that I develop and/or run:



  • MUSIC Code: MUlti-dimensional Stellar Implicit Code
  • MHDT Code: MHD turbulence (incompressible)
  • CSMHD Code: MHD turbulence (compressible)
  • JOREK Code: MHD instabilities in a realistic tokamak geometry (compressible)

Open-source codes that I work with in relation to the codes I develop, and for student projects:



  • MESA Code: Modules for Experiments in Stellar Astrophysics, a 1D stellar evolution code
  • ESTER Code: Evolution STEllaire en Rotation
  • Rayleigh Code: Anelastic MHD convection, pseudospectral code for stellar interiors
  • K-ATHENA Code: finite volume magnetohydrodynamics code
  • NEK5000 Code: A Spectral Element code for CFD
